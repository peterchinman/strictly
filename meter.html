Meter Validator

Instructions

Input a meter using x and /.

x for unstressed syllables.

/ for stressed syllables.

Spaces are ignored.

You can use parentheses to indicate optional parts of the meter.

E.g. / x (x /) could be matched by either / x or / x x /.
